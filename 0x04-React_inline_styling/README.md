# Project: 0x04. React inline styling

React inline styling is a project based on styling React components using inline styling. The project is divided into 4 tasks. The first task is to create a new component named `CourseListRow.js` that returns a row of the table. The second task is to install Aphrodite and create a new component named `BodySectionWithMarginBottom.js` that returns a `div` element with a `h2` element and a `div` element. The third task is to create a new component named `NotificationItem.js` that returns a `li` element. The fourth task is to create a new component named `Notifications.js` that returns a `div` element and a `ul` element.
